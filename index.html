1. Naam aanpassen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Vervang bovenaan de pagina "Naam van student" door jouw eigen naam (id=naam)
  • Commit deze wijziging

2. Afbeeldingen toevoegen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Zoek een passende afbeelding voor dit hoofdstuk (id=afbeeldingen), voeg deze afbeelding toe in de map images.
  • Pas de src aan van de afbeelding van dit hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is.
  • Commit deze wijziging
  • Ga terug naar de master branch.
  • Merge de aanpassingen van afbeeldingen naar master.
  • Push de master branch naar github

3. Een conflict oplossen

Winnaar: Elon Musk

Elon Musk

  • Begin een nieuwe branch: elon vanuit master
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Elon Musk
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/elon-musk.jpg
  • Commit deze wijziging.

Jeff Bezos

  • Begin een nieuwe branch: jeff vanuit master
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Jeff Bezos
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/jeff-bezos.jpeg
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de master branch
  • Refresh de pagina en check of Niemand de winnaar is
  • Merge de aanpassingen van jeff naar master.
  • Controleer of Jeff Bezos de winnaar is (refresh de pagina).
  • Merge de aanpassingen van elon naar master.
  • Conflict!

    Je hebt nu een conflict in de header van dit article.

  • Zorg dat Elon Musk de winnaar is.
  • Commit de merge
  • Push de master branch naar github

4. Een conflict oplossen

Soms is er bij een conflict geen keuze, maar moet je handmatig de wijzigingen samen voegen.

Donaties: DOGE 233,50

Gulle gever

  • Begin een nieuwe branch: brenninkmeijer vanuit master
  • De familie Brenninkmeijer heeft net 233 hele DogeCoin in onze Wallet gestopt.
    Pas het hierboven aangegeven donatiebedrag (id=donaties) aan naar: DOGE 233,00
  • Commit deze wijziging.

Grutte

  • Begin een nieuwe branch: grutte vanuit master
  • Grutte is een beetje skeer, hij heeft 1000 DogeCoin belooft, maar geeft ons slechts een halve.
    Pas het hierboven aangegeven donatiebedrag (id=donaties) aan naar: DOGE 0,50
  • Commit deze wijziging.

Conflict!

  • Merge de aanpassingen van grutte naar master.
  • Controleer of het donatie bedrag klopt (refresh de pagina).
  • Merge de aanpassingen van brenninkmeijer naar master.
  • Conflict!
  • Zorg dat de donaties bij elkaar worden opgeteld.
  • Controleer of het donatiebedrag correct is (refresh de pagina).
  • Commit de merge
  • Push de master branch naar github

5. Een branch rebasen

I are: Coyote

  • Zorg dat je in de master branch zit
  • Begin een nieuwe branch: rebasing
  • Pas de inhoud van het element hierboven (id=animal) aan van Coyote naar: Roadrunner
  • Commit deze wijziging.

No Span(g)

  • Ga terug naar de master branch
  • Pas de tag hierboven aan (id=animal) aan van een span naar een strong
  • Commit deze wijziging.

Rebase!

  • Ga terug naar de rebasing branch
  • Rebase deze branch bovenop de master branch
  • Los het conflict op, controleer het resultaat en maak een commit.
    In het eind resultaat heb je dan een strong met daarin Roadrunner
  • Merge de aanpassingen van rebase naar master.
  • Push de master branch naar github

6. Versie update

Versiebeheer eindigt natuurlijk altijd met een nieuwe versie.

Opdracht

Zorg dat je in de master branch zit.

  • Verhoog de versie (id=version) met 0.1
  • Commit en push de wijziging
  • Lever de URL van jouw github repository in.
